# Product Architecture

AlgoWhirl is built using modular infrastructure that separates trading execution, settlement, and incentive layers.

## Architecture Layers

Frontend

Web trading interface and analytics dashboards.

Orderly Network

Handles the order book, matching engine, and deep liquidity routing.

Settlement Layer

Smart contracts manage settlement, reward accounting, and token distribution.

Indexer

Off‑chain services calculate trading volume, reward points, and referral metrics.

Token Economy

AWT powers incentives, governance, and value capture.

## Design Principles

High performance

Leverages professional trading infrastructure.

Permissionless access

Anyone can trade and participate.

Incentive alignment

Traders and protocol grow together.

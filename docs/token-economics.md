# Token Economics

AlgoWhirl introduces a sustainable token economy where trading activity directly drives token demand.

## Value Capture

Protocol revenue comes from trading fees.

These fees are distributed to:

• Market maker incentives
• Referral rewards
• AWT buybacks

## Buyback Logic

Protocol buybacks purchase AWT from the market using trading fees.

Purchased tokens are distributed as:

50% burned

50% added to liquidity pools

This mechanism simultaneously reduces circulating supply and deepens liquidity.

## Incentive Alignment

The protocol aligns incentives between traders and long‑term holders.

Traders

Earn AWT through trade mining.

Long term participants

Lock AWT into veAWT.

Protocol growth

Increases token demand via buybacks.

This creates a balanced token economy between emission and value capture.

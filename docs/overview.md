# Overview

AlgoWhirl is a perpetual futures trading protocol designed to transform trading activity into tokenized ownership.

Built on top of Orderly Network infrastructure, AlgoWhirl combines deep liquidity with an incentive-driven token economy.

Key Features

- Deep liquidity powered by Orderly Network
- Low trading fees
- Trade mining rewards
- Referral incentives
- Community driven growth

Vision

Trade Perps. Earn AWT.
